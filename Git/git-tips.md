# Git 쓸때 알야아 하는 것들
> git 쓰면서 자주 사용한다 싶은것들 모아놓음

### 1. git 데이터 땡겨오기

```
git init
git remote add origin (해당 repository 위치 .git)

git pull origin (브랜치)



```

* git clone 을 사용해도 좋지만 pull로 땡겨오면 나중에 branch 꼬일일이 없음.

### 2. git push 하기

```
git init
git status
git add (파일)
git commit -m "커밋내용"

git push origin (브랜치)


```
